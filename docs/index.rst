Welcome to MoDiSc's documentation!
===================================

**MoDiSc** is a Python library for debris disks lovers.
MoDiSc (Modeling Disks in Scattereed light) constraints the morphology and photometry of debris disks detected in scattered light. It uses MCMC or Nelder-Mead optimisation to determine the best parameters to fit the disk.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
